# KoalaTree — Konzept & Vision

> *Setz dich hin. Atme tief ein. Der alte Koala im Baum hat eine Geschichte nur fuer dich.*

---

## 1. Big Picture

**KoalaTree** ist eine KI-gestuetzte Plattform fuer personalisierte, paedagogisch wertvolle Audio-Geschichten. Im Kern sitzt ein alter, weiser Koala in einem magischen Eukalyptusbaum — der beste Freund, den man nie hatte. Er kennt dich. Er erinnert sich. Und er erzaehlt Geschichten, die genau fuer dich gemacht sind.

### Was KoalaTree NICHT ist
- Kein generisches KI-Tool das Texte ausspuckt
- Keine Einweg-Geschichte die man einmal hoert und vergisst
- Kein reines Kinderprodukt

### Was KoalaTree IST
- Ein **persoenlicher Begleiter** der mit dir waechst
- Ein **Gedaechtnis** das sich an jede Geschichte erinnert
- Eine **Beziehung** zwischen Zuhoerer und Erzaehler, die ueber Jahre aufgebaut wird
- Am Ende: ein **Audio-Hoerspiel** — warm, meditativ, persoenlich

### Zielgruppen

| Phase | Zielgruppe | Fokus |
|---|---|---|
| **MVP (jetzt)** | Kinder 3-12 | Gute-Nacht-Geschichten, Selbstbewusstsein, Mut |
| **Phase 2** | Teenager 13-17 | Identitaet, Druck, Veraenderung, Mentor auf Augenhoehe |
| **Phase 3** | Erwachsene 18+ | Meditation, Reflexion, inneres Kind, Achtsamkeit |

Der Koala bleibt derselbe — warm, weise, ohne zu urteilen. Nur der Ton und die Tiefe aendern sich.

## 2. Die Story-Welt

### Der KoalaTree
Ein uralter, magischer Eukalyptusbaum. Er leuchtet von innen mit warmem, goldenem Licht. Gluehwuermchen schweben um seine Aeste. Wer sich unter den Baum setzt, wird ruhig. Wer zuhoert, bekommt eine Geschichte — seine eigene.

### Die Zuhoerer
Jeder Mensch der zum Baum kommt, sitzt in seiner eigenen **leuchtenden Blase** — sanft, transluzent, warm. In jeder Blase schweben zarte Story-Elemente (ein Schiff, ein Schmetterling, ein Berg). Goldene Lichtfaeden verbinden jede Blase mit dem Baum. Man ist **gemeinsam** am Baum, aber **allein** in seiner Geschichte.

### Warum ein Koala?
- Koalas strahlen **Ruhe** aus — sie sind die meditativsten Tiere
- Sie sitzen **in Baeumen** — erhoehte Perspektive, Ueberblick, Weisheit
- Sie sind **nicht bedrohlich** — kein Loewe, kein Baer
- Sie sind **universell beliebt** — ueber Kulturen und Altersgruppen hinweg
- Der Eukalyptusbaum ist ihr **Zuhause** — natuerlich, verwurzelt, bestaendig

## 3. Die KoalaTree-Familie

Sechs Charaktere bewohnen den KoalaTree. Jeder hat eine eigene Persoenlichkeit und deckt andere Themen ab.

### Koda — Der Weise Erzaehler (Hauptcharakter)

| | |
|---|---|
| **Rolle** | Haupterzaehler, Stimme des KoalaTree |
| **Fuer** | Alle Altersgruppen |
| **Themen** | Weisheit, Geborgenheit, Gute-Nacht, Lebenslektionen |
| **Visuell** | Silbergraues Fell, warme braune Augen, geflochtenes Armband mit Bernstein |
| **Stimme** | Warm, tief, grossvaeterlich — wie ein alter Freund der am Lagerfeuer erzaehlt |
| **Erkennbar an** | Sitzt immer zentral im Baum, goldene Aura |

Koda beginnt und beendet **jede** Geschichte. Er ist die Konstante.

### Mika — Der Mut-Koala

| | |
|---|---|
| **Rolle** | Abenteuer-Coach, Resilienz-Begleiter |
| **Fuer** | 6-14 Jahre |
| **Themen** | Herausforderungen meistern, Tapferkeit, Angst ueberwinden |
| **Visuell** | Braun-graues Fell, gruenes Bandana, kleiner Kratzer am Ohr |
| **Erkennbar an** | Schaut immer nach draussen, aufmerksame Haltung, bereit zum Aufbruch |

Mika erzaehlt Geschichten ueber Huerden, Abenteuer und die Staerke die in jedem steckt.

### Luna — Die Traum-Koala

| | |
|---|---|
| **Rolle** | Traum-Begleiterin, Meditations-Guide |
| **Fuer** | 3-10 Jahre + Erwachsene |
| **Themen** | Entspannung, Schlaf, Meditation, sanfte Traumreisen |
| **Visuell** | Silber-lavendelfarbenes Fell, Mondform auf der Stirn, Gluehwuermchen um sie herum |
| **Erkennbar an** | Halb geschlossene Augen, schwebt manchmal, immer kuschelig |

Luna ist die sanfteste Stimme. Ihre Geschichten sind wie warme Decken.

### Pip — Der Sternen-Koala

| | |
|---|---|
| **Rolle** | Neugier-Funke, Entdecker |
| **Fuer** | 3-8 Jahre |
| **Themen** | Kreativitaet, Staunen, Entdeckung, Fantasie |
| **Visuell** | Goldbraunes strubbeliges Fell, ueberdimensionale Ohren, Stern-Anhaenger |
| **Erkennbar an** | Immer in Bewegung, haengt kopfueber, breites Grinsen, Funken folgen ihm |

Pip macht alles zum Abenteuer. Seine Geschichten sprühen vor Fantasie und Lebensfreude.

### Sage — Der Stille Koala

| | |
|---|---|
| **Rolle** | Stiller Mentor, Reflexions-Begleiter |
| **Fuer** | 13+ Jahre / Erwachsene |
| **Themen** | Achtsamkeit, innere Ruhe, Selbstreflexion, schwierige Lebensphasen |
| **Visuell** | Dunkelgraues Fell, silberne Straehne, Eukalyptusblatt hinter dem Ohr |
| **Erkennbar an** | Minimalistisch, keine Accessoires, geerdete Meditationshaltung |

Sage redet wenig, aber jedes Wort sitzt. Er ist fuer die Momente, in denen man nicht unterhalten, sondern verstanden werden will.

### Kiki — Der Freche Kookaburra (NEU)

| | |
|---|---|
| **Rolle** | Kodas beste Freundin, Comic Relief, Sidekick |
| **Fuer** | Alle Altersgruppen |
| **Themen** | Humor, Lebensfreude, Begeisterung, andere Perspektive |
| **Visuell** | Golden-braunes und weisses Gefieder, freches Grinsen, gruenes Blatt hinter dem Ohr |
| **Stimme** | Hell, energisch, enthusiastisch — wie eine lustige beste Freundin |
| **Erkennbar an** | Sitzt neben Koda, flattert aufgeregt, lacht viel |

Kiki ist ein Kookaburra (Lachvogel) und lebt im KoalaTree. Sie unterbricht Koda spielerisch, uebernimmt aufregende Teile der Erzaehlung und bringt Humor ohne jemals gemein zu sein. Zusammen mit Koda modelliert sie eine gesunde Freundschaft — verschiedene Persoenlichkeiten die sich ergaenzen.

## 4. Geschichten-Architektur

### Story-Struktur (gilt fuer ALLE Formate)

```
┌─────────────────────────────────────────────┐
│  1. KOALA-INTRO (2-3 Saetze)               │
│     Der Koala begruesst das Kind/den Hoerer │
│     beim Namen. Warm, persoenlich.          │
│     "Hallo [Name]... schoen, dass du da     │
│     bist. Komm, setz dich zu mir..."        │
├─────────────────────────────────────────────┤
│  2. UEBERGANG                               │
│     Die Stimme wird weicher, der Hoerer     │
│     "gleitet" in die Geschichte hinein.      │
├─────────────────────────────────────────────┤
│  3. DIE GESCHICHTE (Hauptteil)              │
│     Personalisiert. Paedagogisch wertvoll.  │
│     Kernbotschaft verpackt in Handlung.     │
│     Angepasst an Alter und Preferenzen.     │
├─────────────────────────────────────────────┤
│  4. KOALA-OUTRO (3-4 Saetze)               │
│     Der Koala kehrt zurueck.                │
│     Ruhiges, weises Fazit.                  │
│     Liebevoll, ermutigend.                  │
│     Sanfte Gute-Nacht-Botschaft.            │
└─────────────────────────────────────────────┘
```

### Story-Formate

| Format | Beschreibung | Koala |
|---|---|---|
| **Traumreise** | Meditative Reise durch magische Welten | Luna / Koda |
| **Weisheitsgeschichte** | Fabel die der Koala "von frueher" kennt | Koda |
| **Dein Abenteuer** | Das Kind ist der Held der Geschichte | Mika |
| **Dankbarkeits-Moment** | Gemeinsamer Rueckblick auf den Tag | Koda / Pip |
| **Stille Reflexion** (Phase 2) | Gefuehrte Meditation / Nachdenken | Sage |

### Altersanpassung

| Alter | Ton des Koalas | Komplexitaet | Beispiel |
|---|---|---|---|
| **3-5** | Sehr sanft, wie ein Grossvater | Einfache Saetze, wiederholend | "Und der kleine Baer legte sich hin... und schlief ein..." |
| **6-8** | Warm, stellt kleine Fragen | Laengere Geschichten, Moral | "Was glaubst du, warum der Fuchs das getan hat?" |
| **9-12** | Respektvoll, philosophisch | Komplexere Themen, Metaphern | "Manchmal ist Mut nicht laut. Manchmal ist Mut ganz still." |
| **13-17** | Mentor auf Augenhoehe | Identitaet, Druck, Veraenderung | Direkte Ansprache, keine Verniedlichung |
| **18+** | Wie ein weiser Freund | Reflexion, Achtsamkeit | Meditative Sprache, offene Fragen |

### Paedagogische Ziele

**Primaer-Fokus:**
- Selbstbewusstsein staerken
- Umgang mit schwierigen Situationen

**Weitere Ziele:**
- Mut und Tapferkeit
- Dankbarkeit und Zufriedenheit
- Empathie und Mitgefuehl
- Achtsamkeit und Praesenz
- Kreativitaet und Fantasie
- Resilienz und innere Staerke

Jedes Ziel wird **nie direkt benannt**, sondern immer in eine Geschichte verpackt. Das Kind sammelt die Botschaften **unterbewusst** — wie Samen die der Koala pflanzt.

## 5. Koala-Gedaechtnis

Das wichtigste Differenzierungsmerkmal: **Der Koala erinnert sich.**

### Wie es funktioniert

1. Bei jeder Story-Generierung werden die letzten 5-10 Geschichten des Kindes geladen
2. Fuer jede wird eine Kurzzusammenfassung gespeichert: Datum, Thema, Kernbotschaft
3. Diese Zusammenfassungen fliessen in den KI-Prompt ein
4. Der Koala referenziert **subtil** vergangene Erlebnisse — nicht jedes Mal, aber wenn es passt

### Beispiele

> *"Erinnerst du dich noch, als wir ueber den mutigen kleinen Fuchs gesprochen haben? Du warst damals gerade in die Schule gekommen. Und jetzt schau dich an..."*

> *"Letztes Mal haben wir zusammen die Sterne gezaehlt. Heute moechte ich dir etwas anderes zeigen..."*

### Warum das wichtig ist
- Schafft eine **echte Beziehung** zwischen Koala und Kind
- Jede Geschichte baut auf vorherigen auf — **Kontinuitaet**
- Das Kind fuehlt sich **gesehen und erinnert**
- Eltern sehen den **Fortschritt** ihres Kindes ueber die Geschichten hinweg
- Langfristiger **Lock-in** — der Koala kennt mein Kind, kein anderes Tool kann das

## 6. Technische Architektur

### Stack

| Komponente | Technologie |
|---|---|
| **Frontend** | Next.js (App Router) |
| **Auth** | Clerk |
| **Datenbank** | Neon (PostgreSQL) via Prisma |
| **KI-Geschichten** | Anthropic Claude API |
| **Audio/TTS** | ElevenLabs API (eleven_multilingual_v2, Multi-Voice) |
| **Sound Effects** | ElevenLabs Sound Generation API |
| **Audio-Speicher** | Vercel Blob |
| **Hosting** | Vercel (Auto-Deploy via GitHub) |
| **Domain** | koalatree.ai |
| **PWA** | Manifest + Media Session API (Background-Audio) |

### Datenmodell (vereinfacht)

```
User (Clerk)
  └── KindProfil (Name, Alter, Interessen, Eigenheiten)
        └── Geschichte (Text, Audio-URL, Format, Ziel, Zusammenfassung)
```

### API-Fluss: Geschichte generieren

```
1. User waehlt Kind + Format + Ziel + optionales Thema
2. Backend laedt Kindprofil + letzte 10 Geschichten (Koala-Gedaechtnis)
3. Claude generiert personalisierte Geschichte mit Koala-Rahmen
4. Text wird gespeichert + Zusammenfassung generiert
5. ElevenLabs wandelt Text in Audio um (Voice: nZpMT2RjIpaat0IaA7Sd)
6. Audio wird in Vercel Blob gespeichert
7. User hoert Geschichte — Text + Audio-Player
```

## 10. Differenzierung & Wettbewerbsvorteil

| Merkmal | KoalaTree | Generische KI-Story-Apps |
|---|---|---|
| **Erzaehler-Charakter** | Konsistenter, wachsender Begleiter | Anonyme KI |
| **Gedaechtnis** | Erinnert sich an jede Geschichte | Jede Geschichte ist isoliert |
| **Audio** | Professionelles Hoerspiel (ElevenLabs) | Text oder simple TTS |
| **Paedagogik** | Unterbewusste positive Botschaften, altersangepasst | Generische Moral |
| **Beziehung** | Waechst mit dem Kind mit, jahrelang | Einweg-Nutzung |
| **Zielgruppe** | 3 bis 99 — ein Begleiter fuers Leben | Meist nur Kinder |

### Der Kern-Satz
> **KoalaTree ist nicht eine App die Geschichten generiert. KoalaTree ist ein weiser Freund der dich ein Leben lang begleitet.**
